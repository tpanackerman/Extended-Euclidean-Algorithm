# Extended-Euclidean-Algorithm

Cài đặt thuật toán Euclidean mở rộng để tìm nghịch đảo nhân của một số trong trường GF(2^10), với đa thức tối giản là m(x) = x^10 + x^3 +1 

Test vector : a = 523, b = 1015 tìm a^-1, b^-1  

In ra từng giá trị trung gian của các bước tính số dư và nhân cho đến khi kết thúc thuật toán
